Class: Bottlebar

Bottlebar()

Represents the bottle status bar in the game UI. Displays the player's current bottle collection progress using different images. Inherits from the Statusbar class.

Constructor

new Bottlebar()

Creates a new Bottlebar instance. Initializes the status bar with a set of images representing different fill levels.
