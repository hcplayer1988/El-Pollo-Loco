Class: ThrowableObject

ThrowableObject(x, y, otherDirection)

Represents a throwable object, such as a salsa bottle, that moves and rotates when thrown. Inherits from MovableObject and includes animation and directional movement.

Constructor

new ThrowableObject(x, y, otherDirection)

Creates a new ThrowableObject instance at the given position and direction. Loads rotation images and initiates the throw animation.
Parameters:
Name Type Description
x number Initial x-coordinate of the object
y number Initial y-coordinate of the object
otherDirection boolean Determines the direction of the throw (true = left, false = right)
